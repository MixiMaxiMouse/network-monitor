# Mini projet de network monitoring

Déstiné a apprendre le network monitoring en python et powershell sous linux.

## V1 : script pour savoir si un process est actif.

- 1 script Python de 50 lignes
- 1 script PowerShell équivalent
- Vérifier si UN service est actif

usage : ```python3 check_service.py ssh``` ou ```pwsh check_service.ps1 ssh```

## V2 : support de plusieurs fichiers
usage: ```python3 check_multiple.py ssh``` ou ```pwsh check_multiple.py ssh```

## V3 : CPU RAM and disk monitor
usage: ```python3 monitor.py``` ou ```pwsh monitor.ps1```

## V4 : generation de rapport HTML
usage: ```python3 report.py && xdg-open rapport.html``` ou ```pwsh report.ps1 && xdg-open rapport.html```
